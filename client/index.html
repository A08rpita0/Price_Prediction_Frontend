Bengaluru Home Price Prediction

This is a data science project where I built a model to predict real estate prices. I used the Bengaluru home prices dataset from Kaggle and implemented a Linear Regression model. The project features a Flask backend that serves HTTP requests using the trained model, and a frontend built with HTML, CSS, and JavaScript where users can input property details. The frontend communicates with the backend to provide real-time price predictions.
